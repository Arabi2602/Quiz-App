🚀 Project Overview: Quiz App with TimerA sleek, interactive, and fully responsive Multiple Choice Question (MCQ) game built using vanilla JavaScript, HTML, and CSS.🛠️ Key FeaturesLive Countdown Timer: Users get exactly 10 seconds per question to think and respond.Instant Visual Feedback: Clicking an answer immediately triggers color changes—Green for Correct and Red for Wrong.Auto-Lock on Timeout: If the timer hits zero, options lock automatically, and the correct answer is highlighted.Dynamic Scoreboard: Tracks performance in real-time and displays the final score at the end with a "Play Again" option.🧠 Core Coding Concepts UsedConceptPurpose in ProjectsetInterval() & clearInterval()Handles the 10-second countdown tick and resets the clock for every new question.State ManagementTracks dynamic data like currentQuestionIndex, score, and timeLeft.DOM ManipulationDynamically injects questions and options into the HTML structure without page reloads.Object ArraysStores the entire quiz dataset (questions, choices, and correct indices) in a clean, structured layout.📂 File Structureindex.html — The skeleton (semantic layout for the quiz interface and results screen).style.css — The skin (clean, professional UI containing color states for active, right, and wrong options).script.js — The brain (handles timer logic, score checking, and application state transitions).
